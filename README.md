# CallDone AI - Script Visualizer

Цей веб-додаток створено для візуалізації скрипту продажів компанії CallDone AI. Він допомагає менеджерам проводити розмови, пропонуючи правильні репліки та варіанти відповідей у зручному інтерактивному форматі.

## Як користуватися

1. Відкрийте файл `index.html` у будь-якому сучасному веб-браузері (Chrome, Safari, Firefox).
2. Введіть ім'я клієнта та ваше ім'я у налаштуваннях.
3. Натисніть "Почати розмову".
4. Читайте текст з картки та обирайте варіанти відповідей клієнта, натискаючи відповідні кнопки.
5. Викорстовуйте кнопку "Крок назад" якщо ви помилилися, або меню історії збоку для орієнтування.

## Структура

- `index.html`: Головна сторінка.
- `style.css`: Стилі та дизайн.
- `script.js`: Логіка додатку.
- `data.js`: Текст скрипту та логіка переходів. Щоб змінити скрипт, редагуйте цей файл.
